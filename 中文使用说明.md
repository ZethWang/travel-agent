# 🌍 AI 旅行规划助手 - 中文使用指南

## 简介
AI 旅行规划助手是一个基于人工智能的智能旅行规划工具，集成了地图导航、天气预报、住宿预订和日历管理等功能，为中国用户提供个性化的旅行规划服务。

## 主要功能

### 🗺️ 智能路线规划
- 自动计算最优路线
- 推荐沿途景点
- 实时交通状况
- 多种交通方式选择

### 🌤️ 天气预报服务
- 目的地实时天气
- 未来一周天气预报
- 天气适宜活动推荐
- 恶劣天气预警

### 🏨 住宿预订助手
- Airbnb 房源搜索
- 价格比较分析
- 用户评价查看
- 预算范围筛选

### 📅 行程日历管理
- 自动创建旅行日程
- 重要事件提醒
- 时区自动转换
- 与 Google 日历同步

## 快速开始

### 1. 环境准备
确保您的系统已安装 Python 3.8 或更高版本。

### 2. 获取必要的 API 密钥

#### Google Maps API
1. 访问 [Google Cloud Console](https://console.cloud.google.com/)
2. 创建新项目或选择现有项目
3. 启用 "Maps JavaScript API" 和 "Places API"
4. 在"凭据"页面创建 API 密钥

#### AccuWeather API
1. 访问 [AccuWeather 开发者中心](https://developer.accuweather.com/)
2. 注册开发者账号
3. 创建新应用获取 API 密钥

#### OpenAI API
1. 访问 [OpenAI 平台](https://platform.openai.com/)
2. 注册账号并充值
3. 在 API 密钥页面创建新密钥

#### Google Calendar API
1. 在 Google Cloud Console 中启用 Calendar API
2. 创建 OAuth 2.0 客户端凭据
3. 生成刷新令牌（需要完成 OAuth 流程）

### 3. 安装和运行

```bash
# 克隆项目
git clone <项目地址>
cd ai_travel_planner_mcp_agent_team

# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run app.py
```

### 4. 配置 API 密钥
启动应用后，在左侧边栏输入所有必需的 API 密钥。

## 使用指南

### 基本操作流程

1. **输入旅行信息**
   - 出发地：输入您的出发城市
   - 目的地：输入您想去的城市
   - 旅行日期：选择出发和返回日期
   - 预算：设置旅行总预算（美元）

2. **选择偏好设置**
   - 旅行风格：冒险、休闲、观光、文化体验等
   - 住宿类型：酒店、青年旅社、公寓、度假村
   - 交通方式：火车、巴士、飞机、租车
   - 饮食要求：素食、纯素、无麸质等

3. **生成旅行计划**
   - 点击"规划我的旅行"按钮
   - AI 代理团队将协作生成个性化计划
   - 查看详细的行程安排和推荐

### 功能详解

#### 地图代理功能
- **路线优化**：计算最短、最快或最经济的路线
- **景点推荐**：根据您的兴趣推荐沿途景点
- **交通建议**：比较不同交通方式的时间和费用
- **实时导航**：提供详细的导航指引

#### 天气代理功能
- **精准预报**：提供目的地详细天气信息
- **活动建议**：根据天气推荐合适的户外活动
- **穿衣指南**：基于天气条件的着装建议
- **预警服务**：及时提醒恶劣天气

#### 预订代理功能
- **智能搜索**：在预算范围内搜索最佳住宿
- **价格对比**：比较不同平台的价格
- **评价分析**：综合用户评价和评分
- **特殊需求**：处理无障碍、宠物友好等特殊要求

#### 日历代理功能
- **行程安排**：自动创建详细的旅行时间表
- **提醒设置**：为重要事件设置提醒
- **时区处理**：自动处理不同时区的时间转换
- **日历同步**：与您的 Google 日历同步

## 常见问题

### Q: API 密钥安全吗？
A: 所有 API 密钥仅在您的浏览器会话中存储，不会上传到服务器。

### Q: 支持哪些城市？
A: 支持全球主要城市，特别针对中国用户优化了国内城市的支持。

### Q: 可以修改生成的行程吗？
A: 当前版本生成固定行程，后续版本将支持手动调整。

### Q: 费用如何计算？
A: 应用本身免费，但需要您自己的 API 密钥，会产生相应的 API 调用费用。
